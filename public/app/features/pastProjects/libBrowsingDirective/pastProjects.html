A website for finding books based on particular characteristics. It takes the average reviews of all users, cycling over each book and averages such characteristics as horror, science fiction content, violence, etc. In Seaching for a book, you select how much of each characteristic you want and the site shows a list with the description of each book matching your search criteria (within a user-chosen range). It is a full MEAN stack app. It uses Angularjs with materialize and CSS for the front-end. For the backend I used; MongoDb, Mongoose, NodeJs, ExpressJs, Passport (Authorization with Google and Facebook), OAuth, and RoboMongo for maintance.
